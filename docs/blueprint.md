# **App Name**: ExpiryGuard

## Core Features:

- Product Addition: Add products to the inventory with name/serial code, and expiration date.
- Inventory Display: View a sortable and searchable inventory list.
- Expiry Notification: Automated notifications for items nearing expiration.

## Style Guidelines:

- Primary color: green (#90EE80) for freshness and reliability.
- Background color: Clean white (#F8F8FF) for clarity.
- Accent color: Amber (#FFBF00) for alerts and warnings related to expiry dates.
- Body and headline font: 'Inter', a sans-serif font, to maintain a modern, easily readable design.
- Use minimalist icons to represent product categories and actions.
- Clean, card-based layout for product listings, ensuring information is easily digestible.
- Subtle transition animations to enhance user experience when navigating the inventory.
